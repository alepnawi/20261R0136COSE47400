# Limitations

- The strongest imported PAnDa result in this repository is still a
  `TruthfulQA sanity-10` development artifact, not a final publication-scale
  benchmark.
- The current implementation was copied from a larger experimental script, so
  it remains more monolithic than a fully polished release package.
- Historical `stage*` names are preserved in the code and metadata for
  provenance, which means some file and preset names are more archival than
  presentation-ready.
- Some experiments depend on external model and dataset availability, so exact
  reproduction may require local caches, Hugging Face authentication, or
  hardware comparable to the original environment.
